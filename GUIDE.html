Всё это на паскале:

---------------------------------

program primer;

label metka;

const a=10;

var x, y :integer;

procedure deystvie();

function deystvie2();

begin

x:=100;
y:=x+a;

end.

---------------------------------
Обратите внимание на точки с запятой
*********************************
program - это название. в данном случае primer. ПРИМЕЧАНИЕ: название не должно начинаться с цифры или содержать русские буквы.
*********************************

label - это описание метки. нужно для прыжка по коду.

НАПРИМЕР:

--------------------------------
program primer2;
label jump;
var a, b :integer;
begin
goto jump;
a:=3;
b:=a*a;
a:=a-b;
jump: Writeln (a);
end.
--------------------------------
Здесь будет выведен ноль, так как команда goto позволяет пропускать код и совершать прыжок в место, где указана метка со знаком двоеточие. То есть, код пропускается и прпограмма продолжает работу на предпоследней строке.
********************************
const - это раздел описания констант, или постоянных. Они НЕ ИЗМЕНЯЮТСЯ В ХОДЕ ПРОГРАММЫ. ИМ НЕЛЬЗЯ ПРИСВАИВАТЬ ЗНАЧЕНИЯ.

НАПРИМЕР, const my = 500 Это выражение значит, что есть постоянная my, которой УЖЕ присвоено значение 500.
********************************
В разделе var описыаются переменные. Пока не присвоить им значение, они равны нулю.
Например, var z :integer - создание переменной z. В этом случае она целочисленна.
